# FURN

## How to install and run locally?

```
$ git clone https://github.com/prachi-sahu-codes/ecomm.git
$ cd ecomm
$ npm install
$ npm start
```

## Features -

- Landing page with furniture categories, hot products and testimonials
- Product listing page
- Filters functionality by:
  - Price
  - Category
  - Ratings
- Sorting functionality by:
  - Relevance
  - Best selling
  - Price: High to Low
  - Price: Low to High
- Search product functionality
- Individual product detail page
- Cart Managment
- Wishlist managment
- Address management
- Order management
- Checkout page
- Payment gateway integration
- Authentication:
  - User Signup
  - User Login
  - logout page
- Error Page
- Toast and Loaders
- Fully Responsive

## Built with -

- React JS
- React Context API + useReducer
- React Router v6
- Vanilla CSS
- React Icons
- Razorpay payment gateway
- mockBee backend
- React Toastify
